## 📚 项目简介

本项目提供了一套高质量的 LaTeX 格式复习笔记，专门针对《毛泽东思想和中国特色社会主义理论体系概论》课程的论述题部分。笔记采用精美的排版设计，结构清晰，重点突出，适合考前复习使用。

### ✨ 特色功能

- **专业排版**：使用 LaTeX 专业排版系统，输出效果精美
- **结构清晰**：采用问题-答案-背诵思路的结构，便于快速查找和记忆
- **重点突出**：关键知识点使用颜色标记，一目了然
- **美观设计**：精心设计的配色方案和版式，阅读体验佳
- **精美封面**：包含精美的封面设计，提升文档质感
- **自动目录**：自动生成目录，方便快速定位
- **易于扩展**：模块化的命令设计，方便添加新的论述题

## 🛠️ 技术栈

- **LaTeX**：文档排版引擎
- **XeLaTeX**：推荐编译引擎（支持中文）
- **ctex**：中文支持宏包
- **tcolorbox**：精美文本框（带 skins 和 breakable 选项）
- **tikz**：绘图和封面设计
- **fontawesome5**：图标支持
- **hyperref**：超链接支持
- **tocloft**：目录格式控制

## 🎨 设计特色

### 配色方案

- **主色调（maincolor）**：深红色 RGB(178, 34, 34) - 体现庄重感
- **次要色（secondarycolor）**：珊瑚色 RGB(255, 127, 80) - 温暖活泼
- **强调色（accentcolor）**：暗红色 RGB(139, 0, 0) - 突出重点
- **背景色（bgcolor）**：暖米色 RGB(250, 245, 240) - 舒适护眼
- **高亮色（softblue）**：软蓝色 RGB(52, 152, 219) - 清晰醒目

### 版式设计

- **精美封面**：使用 TikZ 绘制的流线型设计，包含背景纹理和装饰元素
- **题目框**：带阴影和圆角的精美文本框，标题悬浮设计
- **答案框**：左侧强调线设计，简洁大方
- **页眉页脚**：包含文档标题、日期和页码，底部有装饰线
- **目录**：自动生成，紧凑美观，可点击跳转

### 排版细节

- **行间距**：1.5 倍行距，阅读舒适
- **字体大小**：正文 12pt，标题分级放大
- **页边距**：上下 2.8cm，左右 2.5cm，留白充足

## 🎯 复习建议

1. **先看目录**：通过目录快速定位需要复习的题目
2. **理解要点**：仔细阅读每个问题的答案要点
3. **背诵思路**：利用提供的背诵思路加深记忆
4. **重点标记**：注意高亮部分，这些是考试重点
5. **反复练习**：尝试用自己的话复述答案内容
